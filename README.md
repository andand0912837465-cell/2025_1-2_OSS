1. OSS의 정의 및 의미
OSS (Open Source Software): 누구나 소스 코드를 자유롭게 보고(공개), 사용, 수정, 배포할 수 있는 라이선스를 만족하는 소프트웨어입니다.
핵심: 단순히 '무료'를 넘어, **코드의 자유(Freedom)**를 보장하는 것을 목표로 합니다.
OSI (Open Source Initiative): 공개 소스 정의(OSD)를 관리하고 인증 마크를 부여하는 비영리 조직입니다.
자유 소프트웨어 vs. 오픈소스: 자유 소프트웨어는 사용자의 자유를 강조하며 철학적 측면이 강하고, 오픈소스는 실용적이고 기술적인 측면을 강조합니다.
